Основные принципы

Разметка

Все элементы начинаются с nu-

На каждый атрибут генерится CSS
через функцию

Генерится только уникальный CSS

Непараметризируемые свойста и свойста текста задаются через атрибут mod

* есть исключения, часть свойств дублируется

Значения атрибутов максимально опираются на стандарты, но упрощают их

Метрические атрибуты

Поддерживаются различные кастомные единицы

  • 1x - базовый размер атрибута
  • 1bw - базовая толщина бордюра
  • 1br - базовая ширина радиуса
  • 1p - базовая величина отступа
  • 1** - добавьте кастомный

Другие параметры тем

Также можно задать базовые значения для следующих свойств:

  • padding - --nu-theme-padding
  • border-radius - --nu-theme-border-radius
  • border-width - --nu-theme-border-width
  • shadow-intensity - --nu-theme-shadow-intensity
  • animation-time - --nu-theme-animation-time

Особенности механизма тем

  • Динамические создание и изменение тем
  • Наследование именованных тем от default-темы
  • Наследование тем от родителя
  • Поддерживают Custom Properties experimental
  • Поддержка HTML-цветов и кастомных цветов experimental
  • Адаптивная интенсивность тени
  • Автоматическая Тёмная Тема!
